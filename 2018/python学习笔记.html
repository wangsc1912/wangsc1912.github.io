numpy 数组取行或取列

这个数组跟MATLAB简直不要太像。

首先得导包，from numpy import *

如果你要取前几行

a=c[0:3,:] 前0,1,2行

b = c[0,2:4] 第0行的第2和第3列不包括第4列，因为列下标也是从0开始的

d = c[2:4,2:4] 取中间的2-4行的2-4列

e = c[0,:] 取第0行所有数据

f = c[:,1] 取第1列所有数据

g = c[::2,::2] 两个冒号后面表示步长为2
